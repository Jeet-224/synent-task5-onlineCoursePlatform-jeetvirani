# 🎓 LMS - Online Course Platform

A full-stack Learning Management System (LMS) that allows users to explore online courses, manage learning content, and interact through a modern web interface.

Built using React, Node.js, Express.js, and MongoDB, this platform provides a scalable and responsive learning environment.

---

## 🚀 Features

### Student Features
- Browse available courses
- View course details
- Responsive user interface
- Secure authentication
- User-friendly navigation

### Admin Features
- Manage courses
- Manage users
- Backend API integration
- Database management

### General Features
- Full-stack architecture
- RESTful APIs
- Responsive design
- Environment-based configuration
- Secure backend structure

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/lms-platform.git
```

### Frontend Setup

```bash
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

## 💻 Running the Project

### Start Frontend

```bash
npm run dev
```

### Start Backend

```bash
cd server
npm start
```

Frontend will run on:

```bash
http://localhost:5173
```

Backend will run on:

```bash
http://localhost:5000
```
